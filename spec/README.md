# spec/ — Configuracoes salvas do PyGWalker

Salve aqui arquivos JSON exportados do PyGWalker para que o grafico
carregue ja com uma visualizacao pre-configurada.

## Como gerar:

1. Abra a aba "Analise Livre" no dashboard
2. Monte um grafico no PyGWalker
3. Exporte a configuracao (icone de save no canto)
4. Salve como spec/dashboard_tb.json
5. Commit e push — na proxima abertura carrega automaticamente
